# 🎮 Python Terminal Shooting Game

A fun terminal-based shooting game built with **Python** using **Object-Oriented Programming (OOP)**. Players battle using a luck-based attack system where each round determines who lands the next shot.

---

## 📖 Overview

This project was created to practice core Python programming concepts while building an interactive console game. The game features random bullet damage, health tracking, and a simple battle system displayed with emojis for a more engaging terminal experience.

---

## ✨ Features

* 🎲 Random luck-based battle system
* 🔫 Random bullet damage
* ❤️ Health tracking for both players
* 🎯 Live battle statistics
* 🏆 Automatic winner detection
* 🎮 Emoji-enhanced terminal interface
* 🔁 Continuous battle until a player is defeated

## 🚀 How to Run

1. Clone the repository:

```bash
git clone https://github.com/<your-username>/python-terminal-shooting-game.git
```

2. Navigate to the project folder:

```bash
cd python-terminal-shooting-game
```

3. Run the game:

```bash
python battle_game.py
```

---

## 🎮 Gameplay

1. The game starts with both players having **40 Health Points (HP)**.
2. Each player is assigned a random bullet damage.
3. Enter a luck number between **1 and 6**.
4. The computer generates its own luck number.
5. The player with the higher luck attacks.
6. The battle continues until one player's health reaches zero.
